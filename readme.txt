== Description ==

Access images or videos directly from MarcomGather, import into Wordpress media gallery, and use them for pages and posts.

== Frequently Asked Questions ==

1. Can I use this plugin without a MarcomGather account?

    You will need a MarcomGather account. However, you can sign up for a free trial [by clicking here](https://marcom.com/experience-free-trial/)
2. What are the suppored file types for this plugin?

    JPG, PNG, GIF, MP4, MOV
3. What editors are supported in WordPress?

    The Gutenberg editor is currently supported.
